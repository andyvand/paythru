About

New to Bitcoin? Learn more about how Bitcoin is used to send money over the internet at We Use Coins or get started now by choosing a wallet. Users in the United States can buy bitcoins with Coinbase.

What?

${sitename} is designed to allow tipping or paying to any URI. Right now, we support websites and email addresses. Support is planned for Twitter, Facebook, IRC, or anything else the community wants.

Why?

Tip bots are great, but the need to first register an account and deposit money with them is cumbersome. ${sitename} was created to remove those barriers by unifying all "tipping" methods into a single interface without requiring pre-registration of any kind.

How?

${sitename} will attempt to notify the URI owner that they've received payment (for example, via WHOIS email addresses). When the owner claims the URI through a published method, we report their deposit address along with the claim status in every request.

Published address verification

  • mailto: authorization code emailed to owner
  • http: "bitcoin." DNS TXT entry containing the Bitcoin address
  • http: "bitcoin" HTML meta tag. For example: <meta name="bitcoin" content="1PAYTHRUo5D3jAzGFbLDzjyN7FrZkbuTYo" />
  • http: page scraping for most occuring "valid" Bitcoin address
  • twitter: address in profile description
  • facebook: (Not yet implemented)
  • irc: registered with gribble bot (Not yet implemented)
