For something a little more advanced, just alter the HTTP Accept header for the address endpoint. We officially serve:

  • HTML when application/html (This is how the our website is served)
  • JSON when application/json
  • XML when application/xml
  • Plaintext address when text/plain

An example with curl: curl -iLH "Accept: application/json" http://${hostname}/taylorg@ideaexcursion.com
